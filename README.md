# **Campr Vulnerable Components Workshop**

## **Running in Local Machine**

---

## Requirements

- JDK 1.7 or 1.8
- A Java IDE

## Steps

- [Optional] Generate Intellij project files:

  ```bash
  ./gradlew idea
  ```

- Run the application

  ```bash
  ./gradlew appRun
  ```
  
- Open the app at http://localhost:8080/

- Run the tests. All tests will pass 

  ```bash
  ./gradlew test
  ```
